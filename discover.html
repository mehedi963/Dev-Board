Question-1: What are the different ways to select an element in the DOM?
Ways to Select an Element in the DOM:
  • getElementById("id"): Selects an element by its ID.
  • getElementsByClassName("class"): Selects elements by class
  • getElementsByTagName("tag"): Selects elements by tag name
  • querySelector("selector"): Selects the first matching element
  • querySelectorAll("selector"): Selects all matching elements
Question-2: What is the difference between innerHTML, innerText, and textContent ?
Difference between innerHTML, innerText, and textContent:
  • innerHTML : Returns or sets the HTML inside an element (renders tags).
  • innerText: Returns or sets only the visible text (ignores hidden elements).
  • textContent: Returns or sets all text, including hidden content (ignores HTML tags).
Question-3: What is event delegation in the DOM?
Event Delegation in the DOM:
  • A technique where a parent element listens for events on its child elements.
  • Useful for handling events efficiently on dynamically added elements.
Question-4: What is event bubbling in the DOM?
Event Bubbling in the DOM:
  • When an event on a child element propagates up to its parent and ancestors.
  • Happens by default but can be stopped using event.stopPropagation().
Question-5: How do you create, add, and remove elements using JavaScript?
Creating, Adding, and Removing Elements using JavaScript:
  • Create: document.createElement("tag") creates a new element.
  • Add: appendChild() adds it as a child, insertBefore() places it before another element.
  • Remove: removeChild() removes a child, remove() deletes an element directly.
